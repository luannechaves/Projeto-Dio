# Projeto-Dio
## Projeto destinado à execução de um Wireframe One-Page de média fidelidade. 

O projeto foca na Now Playing Screen (Tela de Reprodução). O design foi criado através de inspirações dos grandes players, extraindo elementos cruciais para criar uma nova interface.
